# Iris-Flower-Classification-Assessment
Iris Flower Classification using Machine Learning::

This project uses the famous Iris dataset to classify iris flowers into three species using Machine Learning.
The model predicts the flower type based on sepal and petal measurements.

Objective:-

The goal of this project is to classify iris flowers into:

Setosa
Versicolor
Virginica
using supervised learning techniques.

Dataset:-

The dataset used is the Iris dataset, loaded from the Seaborn library.

It contains:

Sepal Length
Sepal Width
Petal Length
Petal Width
Species (Target Variable)
Technologies Used:-

Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
Jupyter Notebook
Project Workflow:-

1️:-Import Libraries

Imported all required libraries for data analysis, visualization, and machine learning.

2️:-Load Dataset

Loaded the Iris dataset using Seaborn.

3️:-Data Cleaning & Preprocessing

Checked missing values
Removed duplicate records
Detected outliers using boxplot
Handled outliers
Checked skewness
Standardized features using StandardScaler
4️:-Exploratory Data Analysis (EDA)

Created visualizations such as:

Boxplot
Heatmap
Scatter Plot
5️:-Model Building

Used Logistic Regression for classification.

6️:-Model Evaluation

Evaluated the model using:

Accuracy Score
Classification Report
Confusion Matrix
7️:-Prediction on New Data

Tested the trained model on new unseen flower data.

Results:-

Achieved excellent accuracy on test data
Perfect classification shown in confusion matrix
Successfully predicted new flower species
Key Insights:-

Petal Length and Petal Width have strong correlation
Setosa is clearly separable from other species
Logistic Regression performs very well on Iris dataset
Conclusion

This project demonstrates a complete machine learning workflow:

Data Cleaning Visualization Model Training Evaluation Prediction

It is a great beginner-friendly classification project.
